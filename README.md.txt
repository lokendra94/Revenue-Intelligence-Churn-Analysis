# Revenue Intelligence & Customer Churn Analysis

## Business Problem
Despite acquiring new customers, revenue is stagnating due to high customer churn.
The goal is to understand churn drivers and evaluate how churn reduction impacts revenue.

## Project Overview
This project is an end-to-end Power BI solution that analyzes:
- Monthly Recurring Revenue (MRR)
- Customer churn patterns
- High-risk customer segments
- Revenue impact of churn reduction using what-if analysis

## Tools Used
- Power BI
- Power Query (ETL)
- DAX
- Star Schema Data Modeling

## Key Insights
- Month-to-month contracts show the highest churn
- Early-tenure customers are most at risk
- Small churn reduction leads to meaningful revenue gains
